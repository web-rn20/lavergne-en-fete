# Photos - Lavergne en Fête

Structure des dossiers pour organiser les photos de l'événement.

## Organisation

```
photos/
├── event/       # Photos de l'événement (décoration, ambiance, moments clés)
├── gallery/     # Galerie générale (à afficher sur le site)
├── guests/      # Photos des invités
└── photobooth/  # Photos du photobooth
```

## Formats recommandés

- **Format** : JPG, PNG, WebP
- **Résolution** : 1920x1080 minimum pour les photos de galerie
- **Taille** : Optimiser les images pour le web (< 500KB idéalement)

## Nommage

Utilisez des noms descriptifs en kebab-case :
- `decoration-salle-principale.jpg`
- `groupe-famille-lavergne.jpg`
- `photobooth-001.jpg`
